# ORB-CAST
Official implementation of ORB-CAST: A PUF-enhanced Cross-domain Signcryption Framework for Large-scale IoD Networks.

## 📖 Paper Introduction
This work proposes a revocable, batch-verifiable, secure cross-domain authentication scheme for heterogeneous UAV networks, featuring hardware-rooted trust, anti-cancellation attack capability, and high edge-side efficiency.
